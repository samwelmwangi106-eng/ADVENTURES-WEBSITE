<h1>Samy Safari Adventures</h1>

<p>
Samy Safari Adventures is a simple safari booking web application built using 
HTML, CSS, and JavaScript. The website allows users to view safari packages, 
fill in booking details, and store their bookings using Local Storage.
</p>

<h2>Project Purpose</h2>
<p>
The goal of this project is to practice front-end web development by creating 
a functional safari booking system that stores and displays booking data dynamically.
</p>

<h2>Features</h2>
<ul>
  <li>View available safari packages</li>
  <li>Book a safari using a form</li>
  <li>Save booking details in Local Storage</li>
  <li>Automatically display saved bookings as cards</li>
  <li>Simple navigation between pages (Home, About, Contact)</li>
</ul>
